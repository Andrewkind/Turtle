Turtle Commands

Write any valid JavaScript code and the turtle will perform your actions.

Commands

  • forward(distance);
        ex: forward(100); // Turtle will go forward 100 steps

  • backward(distance);
        ex: backward(100); // Turtle will go backward 100 steps

  • right(angle);
        ex: right(45); // Turtle will rotate right 45 degrees

  • left(angle);
        ex: left(45); // Turtle will rotate Left 45 degrees

  • circle(radius);
        ex: circle(60); // Turtle will draw a circle of 60 pixel radius

  • circle(radius);
        ex: circle(60); // Turtle will draw a circle of 60 pixel radius

  • penUp();
        ex: penUp(); // Turtle will not draw

  • penDown();
        ex: penDown(); // Turtle will draw

  • size(width);
        ex: size(6); // Turtle will draw at width of 6 pixels

  • penSpeed(speed);
        ex: penSpeed(100); // Turtle will draw at the speed of 100 pixels per second

  • color(color);
        ex: color("blue"); // Turtle will draw in blue
        ex: color("#ff0000"); // Turtle will draw in red
